<How websites work?>
1. Browser
2. Through internet, makes a request to the server located anywhere in the world (Complex in detail)
3. Ask for data  (http/https(secured encrypted version of communication)) (http/https protocol)
4. Server > Respond 
    --> 3 files 
           html: text/content of the websites) (Neccessary )
            css: contains all the styling of the website e.g. colors, fonts, position..
            js: give website behavior e.g. login, tweet
5. Display through browser 
** Server can be built by Python :D -> but not neccessary, can be done by java, php, c...
